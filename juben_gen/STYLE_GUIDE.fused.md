# 融合风格约束（基于两套优秀样例）

## 结构指标（1-2分钟/集）
- **scenes_per_ep**：建议 1.7，范围 [1, 3]
- **total_lines_per_ep**：建议 28.1，范围 [22, 38]
- **dialogue_lines_per_ep**：建议 10.28，范围 [10, 20]
- **stage_lines_per_ep**：建议 14.23，范围 [8, 20]
- **vo_os_lines_per_ep**：建议 4.34，范围 [0, 6]

## 格式规范（必须可导出docx且可被脚本校验）
- 集标题：`第{ep}集`
- 场次行：`{ep}-{scene}场  {place}\t{day_or_night}\t{in_or_out}`（用 Tab 或空格分隔也可，但字段顺序别乱）
- 人物行前缀：`人物：`（用顿号“、”分隔）
- 动作/镜头行前缀：`▲`（短句、强视觉）
- 台词行：`{角色名}：{台词}`（中文全角冒号“：”）
- 允许的转场标记：['【切】', '【转】', '【闪回】', '【闪出】']

## 节奏硬规则（来自注意事项）
- 开头30秒抛冲突；每10秒至少一个记忆点（冲突/信息/情绪/动作）。
- 每集：1个核心冲突 + 1个小反转/新信息 + 1个爽点/共情点 + 结尾强钩子。
- 钩子在后续1-2集内回收，同时再埋新钩子，避免挖坑不填。

## 结尾钩子四选一（必须明确到“最后一镜/最后一句”）
- 冲突卡点钩 / 信息反转钩 / 危机升级钩 / 情感抉择钩
