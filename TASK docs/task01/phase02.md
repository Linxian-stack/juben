# Phase 02：红果风格创作规则

## 目标
按创作要素分类，综合提炼红果风格创作规则手册，包含正面规则和反面案例。

## 任务清单

- [ ] 阅读 `docs/小说改编短剧（节奏适配关键注意事项）.docx` 提取节奏规则
- [ ] 阅读 `docs/设计短剧每集结尾的核心.docx` 提取钩子规则
- [ ] 阅读 `docs/短剧一卡通用模板 (2).docx` 提取结构规则
- [ ] 分析样例剧本中的实际表现，验证规则有效性
- [ ] 总结生成剧本过程中的经验教训
- [ ] 编写 `docs/红果风格创作规则.md`，按要素分章：
  - 冲突设计（规则 + 反面案例）
  - 节奏控制（规则 + 反面案例）
  - 钩子机制（4种钩子类型 + 回收策略）
  - 转场技巧（规则 + 反面案例）
  - 情绪曲线（规则 + 反面案例）
  - 付费卡点布局

## 输入
- `docs/` 目录下3份指导文档
- `juben_gen/STYLE_GUIDE.fused.md`

- 样例剧本 + 生成剧本

## 产出

## 验收
