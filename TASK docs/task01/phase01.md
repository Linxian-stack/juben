# Phase 01：样例分析方法论

## 目标
编写详细操作手册，记录如何从样例剧本提取风格画像的统计方法，框架支持未来新增样例。

## 任务清单

- [ ] 阅读 `juben_gen/style_profile.py` 提取统计逻辑
- [ ] 阅读 `juben_gen/style_profile.json` 理解输出结构
- [ ] 阅读两套样例剧本（末世天灾80集、地狱游戏24集）理解数据来源
- [ ] 编写 `docs/样例分析方法论.md`，包含：
  - 统计方法概述（为什么用样例驱动约束）
  - 指标定义（场景数、总行数、台词行、舞台指示行、旁白行）
  - 提取流程（代码示例 + 操作步骤）
  - 数据融合规则（多套样例如何合并）
  - 扩展指南（如何接入新样例）

## 输入
- `juben_gen/style_profile.py`
- `juben_gen/style_profile.json`
- `scripts/末世天灾，我靠吞金超市躺赢.docx`
- `scripts/地狱游戏：开局获得预知羊皮纸1216(1).docx`

## 产出
- `docs/样例分析方法论.md`

## 验收
- 文档包含完整的统计方法说明和代码示例
- 框架设计支持未来新增样例无需改文档结构
