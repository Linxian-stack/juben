# Phase 03：格式规范标准

## 目标
编写剧本格式规范文档，覆盖文本格式和 DOCX 导出规范，附带校验规则文字描述。

## 任务清单

- [ ] 阅读 `juben_gen/STYLE_GUIDE.fused.md` 提取已有格式定义
- [ ] 阅读 `juben_gen/docx_io.py` 理解 DOCX 输出逻辑
- [ ] 分析样例剧本的实际格式细节
- [ ] 编写 `docs/剧本格式规范.md`，包含：
  - 文本格式规范
    - 集标题格式：`第{ep}集`
    - 场次行格式：`{ep}-{scene}场  {place}\t{day_or_night}\t{in_or_out}`
    - 人物行格式：`人物：`（顿号分隔）
    - 动作行格式：`▲`（短句、强视觉）
    - 台词行格式：`{角色名}：{台词}`（中文全角冒号）
    - 转场标记：`【切】【转】【闪回】【闪出】`
  - DOCX 导出规范
    - 字体/字号/行距设定
    - 标题层级样式
    - 段落格式
  - 校验规则描述（供 Phase 4 task04 实现参考）
    - 格式合规检查项
    - 行数范围检查项
    - 比例检查项

## 输入
- `juben_gen/STYLE_GUIDE.fused.md`
- `juben_gen/docx_io.py`
- 样例剧本 DOCX

## 产出
- `docs/剧本格式规范.md`

## 验收
- 文本格式和 DOCX 导出规范完整覆盖
- 校验规则描述足够清晰，可直接用于编码实现
