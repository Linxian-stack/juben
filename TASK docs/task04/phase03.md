# Phase 03：样例对比评估

## 目标
实现生成剧本与优秀样例的统计指标对比，输出表格式对比报告。

## 任务清单

- [ ] 创建 `juben_gen/evaluator.py` 模块
- [ ] 实现统计提取（复用 style_profile.py 逻辑）：
  - 从生成剧本中提取统计指标
  - 加载样例均值（style_profile.json 的 target）
- [ ] 实现对比计算：
  - 对每个指标计算：生成值、样例均值、差值、是否在合理范围内
- [ ] 输出表格式报告：
  ```
  | 指标           | 生成值 | 样例均值 | 范围      | 状态 |
  |----------------|--------|----------|-----------|------|
  | 总行数/集      | 26.5   | 28.1     | [22, 38]  | ✅   |
  | 台词行/集      | 8.2    | 10.28    | [10, 20]  | ⚠️   |
  | 舞台指示行/集  | 15.1   | 14.23    | [8, 20]   | ✅   |
  | 旁白行/集      | 3.2    | 4.34     | [0, 6]    | ✅   |
  | 场景数/集      | 1.8    | 1.7      | [1, 3]    | ✅   |
  ```
- [ ] 保存报告：`{output_dir}/evaluation_report.md`
- [ ] CLI 集成：generate 命令结束后自动输出评估报告

## 输入
- 生成剧本
- `juben_gen/style_profile.json`

## 产出
- `juben_gen/evaluator.py`
- 输出评估报告 MD

## 验收
- 统计指标对比准确
- 超出范围的指标有警告标记
- 报告可读性好
